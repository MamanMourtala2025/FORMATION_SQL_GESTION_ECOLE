# FORMATION_SQL_GESTION_ECOLE
Scripts SQL de la formation SQL Débutant avec PostgreSQL et VS Code. Ce dépôt contient les scripts utilisés dans la vidéo, incluant la création de bases de données, la création de tables, l'insertion de données et les premières requêtes SQL (SELECT, WHERE, ORDER BY, LIMIT). Idéal pour pratiquer et reproduire les démonstrations de la formation.
